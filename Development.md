# Development Guide

## Tech Stack

- Frontend: Angular.js
- Backend: Node.js w/ Express
- Database: PostgreSQL image

## Launch app in dev
Before running anyhing, run ```npm install``` in both frontend/ and backend/

- ```docker compose up --build```
- ```npm run migrate:up``` from backend
- ```node index.js```
- Backend is now running
- ```ng serve```
- Frontend is now running
